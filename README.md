## Project Description
This C++ application helps electrical technicians and students estimate load current, suggest appropriate cable sizes, calculate approximate voltage drop, and generate advisory reports for electrical installations.

## Features
	- Input multiple load circuits
	- Validate user inputs
	- Calculate load current (I = P/V)
	- Calculate voltage drop
	- Recommend cable sizes based on current ranges
	- Determine voltage drop status (Acceptable/High)
	- Generate and save detailed reports
